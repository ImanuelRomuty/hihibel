Login pakai email & password yang dibuat di Supabase — bukan akun GitHub.

Lupa password?

Kami kirim link reset ke email kamu.

Belum bisa login? Buat akun admin dulu:

  1. Buka supabase.com → project hihibel
  2. Authentication → Users
  3. Klik Add user → Create new user
  4. Isi email & password → centang Auto Confirm User
  5. Klik Create user
  6. Login di sini dengan email & password yang sama
